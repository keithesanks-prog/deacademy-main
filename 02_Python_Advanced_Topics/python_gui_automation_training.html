1. Tkinter Fundamentals

What is Tkinter?

  • Python's standard GUI library (built-in, no installation needed)
  • Cross-platform (Windows, Mac, Linux)
  • Widget-based (buttons, labels, text boxes, etc.)

2. SQLGlot Parsing

What is SQLGlot?

  • A SQL parser and transpiler library
  • Converts SQL text ‚Üí Abstract Syntax Tree (AST)
  • Supports multiple SQL dialects

How It Works

import sqlglot

# Parse SQL into AST
sql = "SELECT name, age FROM users WHERE age > 18"
parsed = sqlglot.parse_one(sql)

# Navigate the AST
print(type(parsed))  # <class 'sqlglot.expressions.Select'>

3. Data Lineage Concepts

What is Data Lineage?

Data lineage tracks the flow of data from source to destination:

Flow: Source Table ‚Üí Transformation ‚Üí Target Column

Example

SELECT 
    first_name || ' ' || last_name AS full_name,
    salary * 1.1 AS new_salary
FROM employees
From Table From Column Transformation To Column
employees first_name first_name || ' ' || last_name full_name
employees last_name first_name || ' ' || last_name full_name
employees salary salary * 1.1 new_salary

üõ†Ô∏è Step-by-Step Build Guide

Step 1: Set Up the Window

import tkinter as tk
from tkinter import ttk

# Create main window
root = tk.Tk()
root.title("SQL Lineage Parser")
root.geometry("900x700")

Step 2: Add Input Area

# Header label
tk.Label(root, text="Enter SQL Query", 
         font=("Arial", 16, "bold")).pack(pady=10)

# Text input box
text_area = tk.Text(root, height=10, width=100, 
                    font=("Courier", 10))
text_area.pack(pady=10, padx=10)

Step 3: Define the Parsing Logic

import sqlglot
from sqlglot.expressions import Column, Table, Select
import pandas as pd

lineage = []  # Global list to store results

def extract_lineage(expr, alias_map=None):
    """Extract column lineage from SQL AST"""
    global lineage
    
    if alias_map is None:
        alias_map = {}
    
    # Process SELECT statements
    if isinstance(expr, Select):
        for proj in expr.expressions:
            tgt_col = proj.alias_or_name
            
            for src_col in proj.find_all(Column):
                from_table = src_col.table
                from_col = src_col.name
                
                lineage.append({
                    'From_Table': from_table,
                    'From_Column': from_col,
                    'To_Column': tgt_col,
                    'Transformation': proj.sql()
                })

Step 4: Create the Parse Button

def parse_sql():
    """Parse SQL when button is clicked"""
    global lineage
    lineage = []
    
    sql = text_area.get("1.0", tk.END).strip()
    
    if not sql:
        return
    
    try:
        parsed = sqlglot.parse_one(sql)
        extract_lineage(parsed)
        df = pd.DataFrame(lineage)
        display_results(df)
    except Exception as e:
        print(f"Error: {e}")

# Create button
tk.Button(root, text="Parse SQL", command=parse_sql,
          font=("Arial", 12, "bold"), 
          bg="#4CAF50", fg="white").pack(pady=10)

Step 5: Display Results in a Table

# Create Treeview
tree = ttk.Treeview(root, 
    columns=('From_Table', 'From_Column', 'To_Column', 'Transformation'),
    show='headings',
    height=12)

# Set column headings
tree.heading('From_Table', text='From Table')
tree.heading('From_Column', text='From Column')
tree.heading('To_Column', text='To Column')
tree.heading('Transformation', text='Transformation')

tree.pack(fill=tk.BOTH, expand=True)

def display_results(df):
    """Display DataFrame in Treeview"""
    for item in tree.get_children():
        tree.delete(item)
    
    for index, row in df.iterrows():
        tree.insert('', 'end', values=list(row))

Step 6: Start the Application

# Start the event loop
root.mainloop()
Important: mainloop() must be the last line in your script. It keeps the window open and responsive.

Subquery Example Advanced

SELECT 
    customer_id,
    full_name,
    total_spent
FROM (
    SELECT 
        c.customer_id,
        c.first_name || ' ' || c.last_name AS full_name,
        SUM(o.total_amount) AS total_spent
    FROM customers c
    JOIN orders o ON c.customer_id = o.customer_id
    GROUP BY c.customer_id, c.first_name, c.last_name
) customer_summary
WHERE total_spent > 1000

Tip: Keep business logic in separate functions from GUI code
‚ùå Bad ‚úÖ Good

    data = entry.get()
    # 50 lines of logic...

    # Processing logic
    return result

    data = entry.get()
    result = process_data(data)
    label.config(text=result)
