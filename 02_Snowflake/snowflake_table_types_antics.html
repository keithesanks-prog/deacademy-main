How to Speak the Language (SQL)

Rob here! So we have these cool concepts, but how do we actually tell the computer to do them?

Example 1: The "Permanent" Table (Default)

Note: You don't even have to say "PERMANENT". It's the default!

1:45 The Setup:
Look at the first few lines:
USE ROLE... USE WAREHOUSE... USE SCHEMA...
This is just like telling the GPS where you are before you start driving. You are telling Snowflake "I am the Boss (Role), using this Computer (Warehouse), working in this Folder (Schema)."

2:00 The Command:
CREATE OR REPLACE TABLE...
Simple, right? This creates the "Ink Pen" version.

Example 2: The "Transient" Table (Cheaper)

Note the extra keyword!

2:15 Spot the Difference:
CREATE OR REPLACE TRANSIENT TABLE...
That one word is magic. It tells Snowflake: "Hey, don't worry about the heavy backup (Fail-Safe) for this one. I want to save some money."

Example 3: The "Temporary" Table (Vanish)

-- Create a temporary table
CREATE OR REPLACE TEMPORARY TABLE TEMP_TABLE
(
    ID INT,
    NAME STRING
);

See that Orange word? That means "Delete me when I log out".

Time Travel: How long does the Ink stay?

Rob here again. One last thing. You can actually control how long Snowflake remembers your data after you delete it.

Setting the Time Machine Dial

Using the ALTER command.

3:30 The Magic Number:
DATA_RETENTION_TIME_IN_DAYS
Think of this as the "Undo Button" history.

For Permanent Tables:
You can set this from 0 to 90 days (if you have Enterprise Edition).
ALTER TABLE PERMANENT_TABLE SET DATA_RETENTION_TIME_IN_DAYS = 90;

For Transient / Temporary Tables:
Sorry! These are cheap for a reason.
You can only set this to 0 or 1 day.
ALTER TABLE TRANSIENT_TABLE SET DATA_RETENTION_TIME_IN_DAYS = 1;
Rob: So... which one should I use?

2:30 90% of the time: Use PERMANENT (The default). Just say CREATE TABLE.
2:45 Saving Money? Use TRANSIENT for big raw data dumps. It saves storage costs because it doesn't have that expensive Fail-Safe backup.
3:00 Just Testing? Use TEMPORARY. It cleans itself up so you don't leave a mess.
Summary:

1. Permanent = Ink (Safe, Forever).
2. Temporary = Foggy Mirror (Gone when you leave).
3. Transient = Post-it (Stays, but no backup).
4. External = Window (Looking at S3).
