Lab: Making Data Searchable üîç

Rob: Alright team! We have concepts, but can we actually DO it?

Here is the mission: Take a file from your laptop -> Put it in Snowflake -> Make it searchable.
Let's go!
1

The Cargo Dock (Upload to Stage)

Before Snowflake can "see" your data, you must move it from your computer to Snowflake's internal storage (a Stage).

  • Click Catalog on the sidebar.
  • Go to Database: EMPLOYEE_DATA > Schema: RAW_DATA.
  • Select Stages and click your stage (e.g., EMPLOYEE_STAGE).
  • Click + Files (Top Right) and upload employee_data.csv.

Verify:

LIST @EMPLOYEE_STAGE;
2

The Warehouse (Load to Raw Table)

Now we move the box from the Dock into the Warehouse. We use the COPY INTO command.

Set Context:

USE DATABASE EMPLOYEE_DATA;
USE SCHEMA RAW_DATA;

Run the Load:

COPY INTO EMPLOYEE_RAW
FROM @EMPLOYEE_STAGE/employee_data.csv
FILE_FORMAT = (
    TYPE = 'CSV'
    FIELD_DELIMITER = ','
    SKIP_HEADER = 1
    FIELD_OPTIONALLY_ENCLOSED_BY = '"'
)
FORCE = TRUE;  /* <-- CRITICAL! */
‚ö†Ô∏è Pro-Tip: The "0 Files Processed" Trap
Snowflake remembers files it has already loaded. If you re-upload the same file to fix a mistake, Snowflake will ignore it!

ALWAYS use FORCE = TRUE during development to force it to reload.
3

The Factory (Transform)

Raw data is messy. Let's move it to a "Transformed" room where we clean it up and do math.

CREATE SCHEMA IF NOT EXISTS EMPLOYEE_DATA.TRANSFORMED_DATA;

CREATE OR REPLACE TABLE TRANSFORMED_DATA.EMPLOYEE_TRANSFORMED AS
SELECT 
    ID, 
    CONCAT(FIRST_NAME, ' ', LAST_NAME) AS FULL_NAME,
    SALARY * 12 AS ANNUAL_SALARY,
    DEPARTMENT
FROM RAW_DATA.EMPLOYEE_RAW;
4

The Showroom (Analyze)

Now the data is shiny! Let's search for insights.

Find Top 10% Earners:

SELECT * FROM (
    SELECT *, PERCENT_RANK() OVER (ORDER BY ANNUAL_SALARY DESC) as rnk 
    FROM EMPLOYEE_DATA.TRANSFORMED_DATA.EMPLOYEE_TRANSFORMED
) WHERE rnk <= 0.1;

Search Engineering Team in USA:

SELECT * 
FROM EMPLOYEE_DATA.TRANSFORMED_DATA.EMPLOYEE_TRANSFORMED 
WHERE DEPARTMENT = 'Engineering' 
  AND COUNTRY = 'USA';
