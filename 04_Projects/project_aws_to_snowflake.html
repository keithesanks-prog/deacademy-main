üöÄ Project: Real-Time Data Ingestion to Snowflake

Goal: Build an end-to-end pipeline that ingests real-time data from an API, streams it through AWS Kinesis, and loads it continuously into Snowflake using Snowpipe.

üèóÔ∏è Architecture Overview

Figure 1: End-to-End Data Flow Architecture

The Pipeline Flow:

  1. Ingest: Postman sends JSON data to API Gateway.
  2. Process: Lambda validates data and sends valid records to Kinesis Data Stream (errors go to S3).
  3. Stream: Kinesis Firehose buffers and delivers data to S3 (Data Bucket).
  4. Load: Snowpipe automatically detects new files in S3 and loads them into Snowflake tables.

üõ†Ô∏è Deployment Steps

1 Setup IAM Roles

Create an IAM policy/role that allows:

  • Lambda: Write to Kinesis Streams and PutObjects to S3 (Error bucket).
  • Firehose: specific access to write to the S3 Data bucket.
  • Snowflake: Access to read from the S3 Data bucket (via Storage Integration).

2 Provision S3 Buckets

Create two buckets (or folders):

  • my-project-data-bucket: Landing zone for valid JSON data.
  • my-project-error-bucket: Destination for invalid records.

3 Configure Kinesis

  • Data Stream: Create input-stream.
  • Delivery Stream (Firehose): Source = input-stream, Destination = my-project-data-bucket.

4 Create Lambda Function

Deploy a Python Lambda to validate and route data:

import json
import boto3

kinesis = boto3.client('kinesis')
s3 = boto3.client('s3')

def lambda_handler(event, context):
    try:
        body = json.loads(event['body'])
        if 'user_id' in body and 'event_type' in body:
            # Valid Data -> Kinesis
            kinesis.put_record(
                StreamName='input-stream',
                Data=json.dumps(body),
                PartitionKey=body['user_id']
            )
            return {"statusCode": 200, "body": "Success"}
        else:
            raise ValueError("Missing fields")
    except Exception as e:
        # Invalid Data -> S3 Error Bucket
        s3.put_object(
            Bucket='my-project-error-bucket',
            Key=f"errors/{context.aws_request_id}.json",
            Body=str(e)
        )
        return {"statusCode": 400, "body": str(e)}

5 Setup API Gateway

Create a REST API with a POST method that triggers your Lambda function.

6 Snowflake Integration

Create Table:

CREATE OR REPLACE TABLE raw_events (json_data VARIANT);

Create Pipe:

CREATE OR REPLACE PIPE event_pipe 
AUTO_INGEST = TRUE 
AS 
COPY INTO raw_events 
FROM @my_s3_stage 
FILE_FORMAT = (TYPE = JSON);
