Time Series, Timedeltas, and Date Manipulation

Introduction

Pandas provides powerful tools for handling date and time data. It allows easy manipulation, extraction, and analysis of dates and times.

Why is this important?
  • Enables time-based analysis (e.g. sales per month).
  • Helps detect trends over time.
  • Supports operations like filtering and shifting data.
Data Types
Type Description
datetime64 Stores date and time values (e.g., 2025-01-01 12:00:00)
Timedelta Represents differences between two dates (e.g., 5 days, 2 hours)
Period Represents time periods (e.g., Jan 2025)
1. Creating Date Objects

Use pd.to_datetime() to convert strings to dates.

import pandas as pd dates = pd.to_datetime(['2025-01-01', '2025-02-15', '2025-03-20']) print(dates)
DatetimeIndex(['2025-01-01', '2025-02-15', '2025-03-20'], dtype='datetime64[ns]', freq=None)
2. Extracting Components (.dt accessor)

You can extract year, month, day, hour, etc. easily.

data = {'Date': pd.to_datetime(['2025-01-01', '2025-02-15'])} df = pd.DataFrame(data) # Extract components df['Year'] = df['Date'].dt.year df['Month'] = df['Date'].dt.month df['Day'] = df['Date'].dt.day print(df)
Date Year Month Day 0 2025-01-01 2025 1 1 1 2025-02-15 2025 2 15
3. Handling Time Data

Works for hours, minutes, and seconds as well.

data = {'DateTime': pd.to_datetime(['2025-01-01 08:30', '2025-02-15 14:45'])} df = pd.DataFrame(data) df['Hour'] = df['DateTime'].dt.hour df['Minute'] = df['DateTime'].dt.minute print(df)
DateTime Hour Minute 0 2025-01-01 08:30:00 8 30 1 2025-02-15 14:45:00 14 45
4. Date Ranges & Timedelta

Generate sequences or add time differences.

# Generate 7 days rng = pd.date_range(start='2025-01-01', periods=3, freq='D') print(rng) # Add 7 Days to a DataFrame column df['Next Week'] = df['DateTime'] + pd.Timedelta(days=7) print(df['Next Week'])
DatetimeIndex(['2025-01-01', '2025-01-02', '2025-01-03'], dtype='datetime64[ns]', freq='D') 0 2025-01-08 08:30:00 1 2025-02-22 14:45:00 Name: Next Week, dtype: datetime64[ns]
5. Filtering by Date

Standard conditional logic works with date strings.

# Filter where date is after Feb 1st filtered = df[df['DateTime'] > '2025-02-01'] print(filtered)
