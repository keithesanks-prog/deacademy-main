üåü The F-String Formula

How to make plain text look like a neat table.

f"{value:format}"
Example: f"{salary:.2f}"

üìè Alignment

< Left ^ Center > Right

üî¢ Numbers

.2f (Fixed) , (Comma)
üëà Left Align (<)

Standard for text. Fills space to the right.

name = "Alice"
# Make "Alice" take up 10 characters of space, aligned left
print(f"Name: {name:<10} |")
Name: Alice      |
üëâ Right Align (>)

Standard for numbers. Fills space to the left.

salary = 50000
# Make number take up 10 chars, aligned right
print(f"Salary: {salary:>10} |")
Salary:      50000 |
üéØ Center Align (^)
role = "Admin"
print(f"|{role:^10}|")
|  Admin   |
üìù The Manual Table Trick: Combining Left and Right alignment creates a clean table manually.
print(f"{'Item':<10} {'Price':>10}")
üí∞ 2 Decimal Places (.2f)

The most important one for money.

val = 123.456789
print(f"Value: {val:.2f}")
Value: 123.46
aaa Thousands Separator (,)

Adds commas to big numbers.

big_num = 1000000
print(f"Cost: ${big_num:,}")
Cost: $1,000,000
ü¶∏ Combo! (,.2f)

Commas AND Decimals together.

money = 1234567.891
print(f"Price: ${money:,.2f}")
Price: $1,234,567.89

# Data
data = [
    ("Laptop", 1200.50),
    ("Mouse", 25.00),
    ("Monitor", 399.99),
]

print("-" * 30)
# 1. Print Header
# "Product" Left aligned (15 width)
# "Price" Right aligned (10 width)
print(f"{'Product':<15} {'Price':>10}")
print("-" * 30)

# 2. Print Rows with Loop
for product, price in data:
    # Print Product Left (15)
    # Print Price Right (10), formatted with 2 decimals
    print(f"{product:<15} {price:>10.2f}")
    
print("-" * 30)
