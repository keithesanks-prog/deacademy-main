üìù What is a Pandas Series?

A Series in Pandas is a one‚Äëdimensional labeled array that can hold data of any type (integers, floats, strings, Python objects, etc.). Each element has a value and an index label.

Key Points:
  • Similar to a list or 1‚ÄëD NumPy array.
  • Provides powerful label‚Äëbased indexing.
  • Can hold heterogeneous data types.

üîß Accessing Series Properties
import pandas as pd

series = pd.Series([5, 10, 15], index=['a', 'b', 'c'])
print(series.index)   # Index(['a', 'b', 'c'], dtype='object')
print(series.values)  # [ 5 10 15]
print(series.dtype)   # int64
Calculations Scalar Operations

You can perform mathematical operations on an entire Series at once.

import pandas as pd

data = [2, 4, 6, 8, 10]
series = pd.Series(data)

# Add 5 to every element
addition = series + 5
print(addition)

# Multiply every element by 3
multiply = series * 3
print(multiply)
# Output (Addition):
0     7
1     9
2    11
3    13
4    15
dtype: int64

# Output (Multiplication):
0     6
1    12
2    18
3    24
4    30
dtype: int64
üìä Aggregation
# Calculate the mean
mean = series.mean()
print(mean)  # Output: 6.0

‚ö° Apply & Replace
data = [10, 20, 30, 40]
