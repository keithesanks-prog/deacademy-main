üîë The Key Function: pd.set_option('name', value)
üìè Max Rows & Max Columns

Force Pandas to show more data instead of truncating with ellipses (...).

import pandas as pd

# Show up to 100 rows
pd.set_option('display.max_rows', 100)

# Show up to 50 columns
pd.set_option('display.max_columns', 50)

# To see EVERYTHING (be careful with big data!):
# pd.set_option('display.max_rows', None)
üîÑ Reset to Defaults

Mess up your view? Go back to normal.

pd.reset_option('display.max_rows')
pd.reset_option('display.max_columns')

# Or reset EVERYTHING
# pd.reset_option('all')
üéØ Float Format (The Money Fix)

The best way to make financial data readable (2 decimal places).

# Standard specific format (2 decimals)
pd.set_option('display.float_format', '{:.2f}'.format)

# Example Output:
# Before: 50000.5
# After:  50000.50
üö´ Scientific Notation

Use the same trick to kill scientific notation (e.g., 1.23e+09).

# This forces normal number display instead of scientific
pd.set_option('display.float_format', '{:.2f}'.format)

df_large = pd.DataFrame({'Value': [0.00012345, 123456789.12345]})
print(df_large)
          Value
0          0.00
1  123456789.12
üìê Max Column Width

Controls how many characters to show inside a single cell before truncating.

# Allow text to be 500 characters long before cutting off
pd.set_option('display.max_colwidth', 500)

# Or restrict it to just 10 chars
pd.set_option('display.max_colwidth', 10)
