Time Travel for Data: Comparing Past & Future

What is Shifting?

Shifting moves data along the time index by a specified number of periods.

Definitions:
  • Lagging (Positive Shift): Moving data down (to later dates). Compare today with yesterday.
  • Leading (Negative Shift): Moving data up (to earlier dates). Compare today with tomorrow.
Common Use Cases
  • Comparing current values with past values ("Day-over-Day change").
  • Calculating percentage growth.
  • Identifying trends and seasonality.
Syntax
# Shift forward by 1 (Yesterday's value moves to Today) df['Lag'] = df['Col'].shift(periods=1) # Shift backward by 1 (Tomorrow's value moves to Today) df['Lead'] = df['Col'].shift(periods=-1) # Fill NaN values created by shifting df['Lag_Filled'] = df['Col'].shift(1, fill_value=0)
Setup Data
import pandas as pd data = { 'Date': pd.date_range(start='2025-01-01', periods=5, freq='D'), 'Sales': [200, 220, 210, 215, 250] } df = pd.DataFrame(data) df.set_index('Date', inplace=True)
1. Basic Shifting (Lag & Lead)
# Lag: Get previous day's sales df['Prev_Day'] = df['Sales'].shift(1) # Lead: Get next day's sales df['Next_Day'] = df['Sales'].shift(-1) print(df)
Sales Prev_Day Next_Day 2025-01-01 200 NaN 220.0 2025-01-02 220 200.0 210.0 2025-01-03 210 220.0 215.0 2025-01-04 215 210.0 250.0 2025-01-05 250 215.0 NaN
2. Calculating Differences

How much did sales change from yesterday?

df['Diff'] = df['Sales'] - df['Sales'].shift(1) print(df[['Sales', 'Diff']])
Sales Diff 2025-01-01 200 NaN 2025-01-02 220 20.0 2025-01-03 210 -10.0
3. Percentage Change

Using pct_change() (shortcut for shift math).

df['Pct_Change'] = df['Sales'].pct_change() * 100 print(df[['Sales', 'Pct_Change']])
Sales Pct_Change 2025-01-01 200 NaN 2025-01-02 220 10.000000 2025-01-03 210 -4.545455
