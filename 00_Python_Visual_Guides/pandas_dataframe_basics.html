1Ô∏è‚É£ From List of Lists

Create a DataFrame by passing a list of lists and specifying column names.

import pandas as pd

# 1. Create DataFrame from list of lists
data = [[101, 'John', 29], [102, 'Anna', 25], [103, 'Peter', 32]]
df = pd.DataFrame(data, columns=['ID', 'Name', 'Age'])

print(df)
    ID   Name  Age
0  101   John   29
1  102   Anna   25
2  103  Peter   32
2Ô∏è‚É£ From Dictionary

Create a DataFrame from a dictionary where keys become column names.

# 2. Create DataFrame from dictionary
data = {'Product': ['A', 'B', 'C'], 'Price': [100, 200, 150]}
df = pd.DataFrame(data)

print(df)
  Product  Price
0       A    100
1       B    200
2       C    150
‚ûï Add Column

Add a new column by assigning a list of values to a new column name.

# Create initial DataFrame
data = [[101, 'John', 29], [102, 'Anna', 25], [103, 'Peter', 32]]
df = pd.DataFrame(data, columns=['ID', 'Name', 'Age'])

# Add a new column
df['City'] = ['NY', 'LA', 'Chicago']
print(df)
    ID   Name  Age     City
0  101   John   29       NY
1  102   Anna   25       LA
2  103  Peter   32  Chicago
üì• Add Row

Use .loc[] to add a new row at a specific index.

# Setup DataFrame
data = {'Product': ['A', 'B', 'C'], 'Price': [100, 200, 150]}
df = pd.DataFrame(data)

# Add a new row at index 3 using a dictionary
df.loc[3] = {'Product': 'D', 'Price': 180}
print(df)
  Product  Price
0       A    100
1       B    200
2       C    150
3       D    180
üóëÔ∏è Delete Row

Use .drop() with axis=0 (rows) and inplace=True to remove a row.

# Setup DataFrame
data = {'Product': ['A', 'B', 'C'], 'Price': [100, 200, 150]}
df = pd.DataFrame(data)

# Delete row at index 2
df.drop(2, axis=0, inplace=True)
print(df)
  Product  Price
0       A    100
1       B    200
