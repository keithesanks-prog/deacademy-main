üí° PRACTICAL EXAMPLES

Example 1: Get last 3 items in reverse

Example 2: Check if list is palindrome

my_list = [1, 2, 3, 2, 1] is_palindrome = my_list == my_list[::-1] print(is_palindrome) # True

Example 3: Reverse a string
