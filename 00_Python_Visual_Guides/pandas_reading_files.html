Master loading CSV and Excel data

üìÑ Reading CSV Files

The workhorse of data science. Here is how to read basic and complex comma-separated files.

import pandas as pd # 1. Basic Read (comma default) df = pd.read_csv('data.csv') # 2. Custom Separator (e.g. semicolon) df = pd.read_csv('data_semicolon.csv', sep=';') # 3. Handling Missing Values (Custom strings -> NaN) df = pd.read_csv('data.csv', na_values=['NA', 'null', ''])
üéõÔ∏è Controlling Columns & Types

Load only what you need, in the format you want.

# Specific Columns + Data Types df = pd.read_csv('data.csv', usecols=['Name', 'Salary'], dtype={'Salary': 'float32'}) # Set Index Column immediately df = pd.read_csv('data.csv', index_col='Name')
üìä Reading Excel Files

Reading .xlsx or .xls files. You can target specific sheets.

# 1. Read First Sheet (Default) df = pd.read_excel('data.xlsx') # 2. Read Specific Sheet (by Name or Index) df = pd.read_excel('data.xlsx', sheet_name='Sheet1') df = pd.read_excel('data.xlsx', sheet_name=1) # 2nd sheet # 3. Select Columns df = pd.read_excel('data.xlsx', usecols=['Name', 'City'])
üõ†Ô∏è Parameter Cheat Sheet

These options work for both read_csv and usually read_excel.

Parameter Description Example
filepath Path to file 'data.csv'
sep Delimiter (CSV only) sep=';' or sep='\t'
header Row # for column names header=0 (Default)
usecols List of columns to read ['Name', 'Age']
index_col Col to use as Row Labels 'ID'
dtype Force data types {'Age': 'int32'}
na_values Strings to treat as NaN ['?', 'MISSING']
encoding File encoding 'utf-8', 'latin1'
