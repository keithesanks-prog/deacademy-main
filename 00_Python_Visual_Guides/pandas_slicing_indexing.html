üîç Selecting Columns

Think of columns like columns in a spreadsheet. You just grab them by name.

Bracket Notation [] (Best)
df['Name']

‚úÖ Works for all column names.
‚úÖ Can select multiple: [['Name', 'Age']]

Dot Notation . (Shortcut)
df.Name

‚ùå Fails if column has spaces.
‚ùå Cannot select multiple.

üìã Selecting Rows (The Basics)

To get specific rows, we usually need specific tools like loc or iloc which we will master in the next tab.

But simpler slicing [start:end] works too!

# Get first 3 rows df[0:3]

üìç The Great Confusion: Labels vs Positions

This is the hardest part for beginners. Let's solve it with an analogy.

1. iloc (Integer Location)

üì° GPS

"Go to coordinate 0."

Uses Positions (0, 1, 2...). It doesn't care what the row is named.

# Row 0 df.iloc[0] # Rows 0 to 2 (Excludes 3!) df.iloc[0:3]

2. loc (Location)

üè† Street Address

"Go to 123 Main St."

Uses Labels (names). If the index is 'Alice', you look for 'Alice'.

# Row labeled 'Alice' df.loc['Alice'] # Rows 'Alice' to 'Charlie' (Inclusive!) df.loc['Alice':'Charlie']
üìê Anatomy of Slicing

df.iloc[ ROWS , COLUMNS ]


  • df.iloc[0, 1] -> Row 0, Column 1
  • df.iloc[:, 0:2] -> : means "All Rows", Columns 0 to 2
Conditional Selection

Filter rows based on true/false conditions.

# Standard Boolean Indexing df[df['Age'] > 30] # Multiple Conditions (AND = &, OR = |) df[(df['Age'] > 30) & (df['City'] == 'Boston')] # Is In List df[df['City'].isin(['Chicago', 'Boston'])]
‚ú® The Query Method (The Readable Way)

Write filters like SQL or plain text.

# So much cleaner! df.query('Age > 30') df.query('Age > 30 and City == "Boston"')
Resetting & Setting Index

The "Index" is the bold left-most column (the labels). You can change it.

Set Index
Turn a column into the index labels.
df.set_index('Name', inplace=True)
Now you can use df.loc['Alice']!
Reset Index
Turn the index back into a standard column (0, 1, 2...).
df.reset_index(inplace=True)
