What is String Slicing?

String slicing lets you extract parts of a string using a simple syntax. Think of it like cutting a piece of cake - you specify where to start cutting, where to stop, and how big each slice should be!

üìù General Syntax
string[start:stop:step]
  • start - Where to begin (inclusive)
  • stop - Where to end (exclusive)
  • step - How many characters to skip

Quick Reference Cheat Sheet

Common Patterns

text[:n] # First n characters
text[n:] # From n to end
text[-n:] # Last n characters
text[:-n] # All except last n
text[::] # Copy entire string
text[::-1] # Reverse string
