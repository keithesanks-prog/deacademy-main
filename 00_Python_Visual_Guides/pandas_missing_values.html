Handling Missing Values üïµÔ∏è‚Äç‚ôÄÔ∏è

Identify, remove, and fill NaN/None values

üåü Why Handle Missing Values?

Real-world data is messy. You must decide: Drop or Fill?

üö´ The Problem

ML models fail. Statistics get skewed. Insights become inaccurate.

üîç Identify

isna() / isnull()

üõ†Ô∏è Fix

dropna() / fillna()
1Ô∏è‚É£ isna() & isnull()

Returns True where the value is missing (NaN) and False otherwise. 

import pandas as pd
import numpy as np

data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'David'],
    'Age': [25, None, 35, 40],
    'City': ['New York', 'Chicago', None, 'Seattle'],
    'Salary': [50000.5, 60000.0, None, 80000.25]
}

df = pd.DataFrame(data)

# Both do the exact same thing
print(df.isna())
# print(df.isnull())

üóëÔ∏è Drop Rows

Remove any row that has at least one missing value.

# Remove rows with ANY missing values
df_cleaned = df.dropna()
print(df_cleaned)
    Name   Age      City   Salary
0  Alice  25.0  New York  50000.5
3  David  40.0   Seattle  80000.2
üèóÔ∏è Drop Columns

Remove any column that has missing values (be careful with this!).

# axis=1 means Columns
df_cleaned = df.dropna(axis=1)
print(df_cleaned)
      Name
0    Alice
1      Bob
2  Charlie
3    David
üßΩ Drop if ALL missing

Only remove a row if every single value in that row is missing.

df.dropna(how='all')
üíâ Fill with Constant

Replace NaN with 0, 'Unknown', or any other fixed value.

df_filled = df.fillna(0)
print(df_filled)
      Name   Age      City   Salary

üìä Fill with Statistics (Mean/Median)

A smart way to fill numerical gaps without skewing data too much.

# Fill Age with Mean
df['Age'] = df['Age'].fillna(df['Age'].mean())

# Fill Salary with Median
df['Salary'] = df['Salary'].fillna(df['Salary'].median())

print(df)
      Name   Age      City   Salary

1      Bob  33.3   Chicago  60000.0
2  Charlie  35.0      NaN  60000.0
3    David  40.0   Seattle  80000.2
‚è≠Ô∏è Forward & Backward Fill

Great for time-series data. Fills with the previous (ffill) or next (bfill) valid value.

df.ffill()  # Propagate last valid observation forward
df.bfill()  # Use next valid observation to fill backward
üìà Interpolate

Estimates values based on a linear trend between existing points.

df['Age'] = df['Age'].interpolate()
