Pandas Data Types (dtypes) üî†

Optimize Memory & Unlock Functionality

Why do types matter?

Pandas often guesses types incorrectly (everything becomes an 'object' or 'string'). Changing them unlocks special powers.

object (String)

The Default. Slow, eats memory. Can't do math or date logic.

category

The Memory Saver. Great for repeating text (City: NY, NY, NY). Saves 90%+ memory!

datetime64

The Time Traveler. Unlocks .dt.year, time slicing, and more.

üõ†Ô∏è The Casting Mold: .astype()

Use .astype() to force a column into a new shape.

# The Syntax df['Column'] = df['Column'].astype('new_type')
Scenario Code Why?
Save Memory .astype('category') Best for columns with few unique values (Gender, Country, Status).
Decimals -> Integers .astype('int64') Remove decimal points (Age: 30.0 -> 30).
Precision Control .astype('float32') Use less memory than default float64.
Time Series .astype('datetime64[ns]') CRITICAL: Converts string dates "2023-01-01" to real Date objects.
