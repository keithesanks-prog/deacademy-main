3. The Dictionary (Lookup)

A dictionary is like a warehouse. You use a Key üîë to find a specific Value üéÅ.

The Setup

The Access

‚ö†Ô∏è Pitfall: The Double Index
Why does user["name"][0] print "A"?
1. user["name"] gets "Alice" (The String)
2. [0] then gets the 1st letter of "Alice" (String Slicing)
Be careful not to slice the string when you meant to just get the value!

7. The Reversal (Dict Comprehension)

A "Comprehension" is a one-line loop. We use it to FLIP keys and values.

Anatomy of the One-Liner:
{ value: key for key, value in my_dict.items() }
  • The Loop (Right Side):
    This part runs first. It iterates through the pair `("a", 1)`, then `("b", 2)`.
  • The Blueprint (Left Side):
    This tells Python how to structure the NEW item.
    "Take the old Value (1) and make it the Key. Take the old Key ('a') and make it the Value."
my_dict = {"a": 1, "b": 2, "c": 3} # The Syntax: { NEW_KEY : NEW_VALUE for ... } # We are saying: "Set the Value as the Key" reversed_dict = {value: key for key, value in my_dict.items()} print(reversed_dict) # Output: {1: 'a', 2: 'b', 3: 'c'}

14. The Bouncer (Filter)

Filter takes a list and keeps only the items that pass a test (True).

names = ['allen', 'eve', 'charlie', 'richmond'] # Goal: Keep only names with EVEN length # We use a Lambda to verify each name on the fly evens = filter(lambda num: len(num)%2 == 0, names) print(list(evens)) # Output: ['richmond'] # (richmond has 8 letters. eve has 3. allen has 5.)

15. The Conveyor Belt (Map)

Map takes a list and transforms every single item into something else.

names = ['allen', 'eve', 'charlie'] # Transformation: Get the length of every name name_lengths = map(len, names) print(list(name_lengths)) # Output: [5, 3, 7]

The "Shadowing" Trap: If you name a local variable the same as a global one, the local one wins inside the function, but the global one remains unchanged outside.

21. The Special Agents (Dunder Methods)

"Dunder" (Double Underscore) methods allow your objects to use standard Python tricks.

Common Agents:
  • __init__: The Constructor (Called on birth).
  • __str__: The Nametag (Called by print()).
  • __len__: The Ruler (Called by len()).
class Book(): def __init__(self, title, pages): self.title = title self.pages = pages # Controls print(book) def __str__(self): return f"Book: {self.title}" # Controls len(book) def __len__(self): return self.pages b = Book("Python 101", 50) print(b) # Output: Book: Python 101 print(len(b)) # Output: 50
