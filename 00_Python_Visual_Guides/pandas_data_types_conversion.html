üîç Check Data Types (`dtypes`)

Always start here. Does Pandas think your numbers are text (object)?

import pandas as pd

data = {
    'Name': ['Alice', 'Bob'],
    'Age': [25, 30],
    'Salary': [50000.5, 60000.0],
    'Join_Date': ['2023-01-10', '2022-11-15']
}

df = pd.DataFrame(data)

# Print the types
print(df.dtypes)
Name          object
Age            int64
Salary       float64
Join_Date     object
dtype: object
Note: In Pandas, object usually means String (text).
1Ô∏è‚É£ astype() - The Generalist

Force a column into a specific type (int, float, str).

# Integer to Float
df['Age'] = df['Age'].astype('float')

# Number to String
df['Age'] = df['Age'].astype('str')
2Ô∏è‚É£ to_numeric() - The Specialist

Best for converting messy strings to numbers. Handles errors gracefully.

# Convert 'Salary' to integer (downcast optimizes memory)
df['Salary'] = pd.to_numeric(df['Salary'], downcast='integer')
3Ô∏è‚É£ to_datetime() - The Time Traveler

Converts date strings into actual Timestamp objects.

# "2023-01-10" becomes a real date
df['Join_Date'] = pd.to_datetime(df['Join_Date'])

print(df.dtypes)
Join_Date    datetime64[ns]
dtype: object

Asks Pandas to just "make it better" automatically.

df = df.convert_dtypes()
