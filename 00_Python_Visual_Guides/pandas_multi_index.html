Pandas Multi-Level Indexing üìö

Organize data with Hierarchical Indexes

üåü The Hierarchy Concept

Think of it like folders on your computer: Folder > Subfolder > File

üìÇ Level 1: Group

Country, Department, Year

üìÅ Level 2: Sub-Group

City, Team, Quarter

üìÑ Values

Sales, Population, Revenue

üí° Why? Creating subgroups makes it incredibly easy to select chunks of data (e.g. "Give me all cities in India").
1Ô∏è‚É£ Method 1: set_index()

The most common way. Take existing columns and turn them into the index.

import pandas as pd

data = {
    'Country': ['USA', 'USA', 'India', 'India', 'Canada', 'Canada'],
    'City': ['New York', 'Chicago', 'Mumbai', 'Delhi', 'Toronto', 'Vancouver'],
    'Population': [8500000, 2700000, 20000000, 18000000, 2900000, 2500000]
}
df = pd.DataFrame(data)

# Create multi-level index using 'Country' and 'City'
df.set_index(['Country', 'City'], inplace=True)

print(df)

2Ô∏è‚É£ Method 2: From Tuples (Advanced)

Create the structure from scratch before making the DataFrame.

index = pd.MultiIndex.from_tuples(
    [('USA', 'New York'), ('USA', 'Chicago'),
     ('India', 'Mumbai'), ('India', 'Delhi'),
     ('Canada', 'Toronto'), ('Canada', 'Vancouver')],
    names=['Country', 'City']
)

data = [8500000, 2700000, 20000000, 18000000, 2900000, 2500000]
df = pd.DataFrame(data, index=index, columns=['Population'])
print(df)
üîÑ Reset Index

Undo the multi-index and return columns to normal.

df_reset = df.reset_index()
print(df_reset)
üéØ Specific Row

Access data by providing the full path (tuple).

# Access population of Mumbai (India -> Mumbai)
print(df.loc[('India', 'Mumbai')])
üìÇ Partial Index (Group Selection)

Select everything under a top-level group.

# Get all data for 'India'
print(df.loc['India'])
