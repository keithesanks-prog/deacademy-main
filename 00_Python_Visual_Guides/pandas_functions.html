üí° Quick Cheat Sheet:
- Use apply() for complex logic on rows/columns.
- Use map() for simple swaps (like A->B) on a Series.
üöÄ The Powerhouse: apply()

Runs a custom function on every single row or column. Very flexible.

# 1. Simple Math using Lambda
df['New_Price'] = df['Price'].apply(lambda x: x * 1.10)
üß† Custom Function Logic

Define a complex python function and apply it to the whole row (axis=1).

def total_value(row):
    # Logic across multiple columns
    return row['Price'] * row['Stock']

# axis=1 means "Apply to each ROW"
df['Total_Value'] = df.apply(total_value, axis=1)
üó∫Ô∏è The Mapper: map()

Great for formatting or simple dictionary lookups. Only works on Series (Columns).

# Format price Strings
df['Price_Str'] = df['Price'].map(lambda x: f"${x}")
üåê The Global: applymap()

Applies a function to every single cell in the entire DataFrame.

# Convert EVERYTHING to string
result = df[['Price', 'Stock']].applymap(str)
üîÑ The Shape Keeper: transform()

Applying a function but guaranteeing the result is the same shape.

# Double the price using transform
df['Double_Price'] = df['Price'].transform(lambda x: x * 2)
