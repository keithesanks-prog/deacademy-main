Pandas Indexing: loc vs iloc üéØ

Deep Dive into Label vs Position Selection

What is Indexing?

Indexing allows you to select specific rows and columns from a DataFrame using either Labels (Names) or Positions (GPS Coordinates).

üè∑Ô∏è loc[]

Label-based indexing.

Think: "Find row named 'Alice'."

üìç iloc[]

Position-based indexing.

Think: "Find row #0 (the first one)."

üè∑Ô∏è loc[]: Label-Based Selection

Selects rows/columns by Name. Includes the end index in slicing.

Syntax: DataFrame.loc[row_label, column_label]
Parameter Description
row_label Label(s) of row(s) to select
column_label Label(s) of column(s) to select
Setup Data:
import pandas as pd data = { 'Name': ['Alice', 'Bob', 'Charlie', 'David'], 'Age': [25, 30, 35, 40], 'City': ['New York', 'Chicago', 'Boston', 'Seattle'] } df = pd.DataFrame(data, index=['a', 'b', 'c', 'd'])

Examples

# 1. Single Row df.loc['a'] # 2. Multiple Rows (List) df.loc[['a', 'c']] # 3. Specific Rows & Cols df.loc[['a', 'c'], ['Name', 'Age']] # 4. Slicing (End INCLUDED!) df.loc['b':'d', 'Name':'Age'] # 5. Conditional df.loc[df['Age'] > 30]
‚úèÔ∏è Updating Values with loc
# Update Age for row 'b' (Bob) df.loc['b', 'Age'] = 32
üìç iloc[]: Position-Based Selection

Selects rows/columns by Integer Index (0, 1, 2...). Excludes the end index in slicing.

Syntax: DataFrame.iloc[row_index, column_index]
Parameter Description
row_index Integer index of row(s)
column_index Integer index of column(s)

Examples

# 6. Single Row (Index 0) df.iloc[0] # 7. Multiple Rows (List of Indices) df.iloc[[0, 2]] # 8. Specific Rows & Cols (Indices) # Rows 0 & 2, Cols 0 & 1 df.iloc[[0, 2], [0, 1]] # 9. Slicing (End EXCLUDED!) # Rows 1 to 3 (so 1 & 2), Cols 0 to 2 (so 0 & 1) df.iloc[1:3, 0:2] # 10. Conditional (Tricky!) # iloc needs simple values or booleans, so we use .values df.iloc[(df['Age'] > 30).values]
‚úèÔ∏è Updating Values with iloc
# Update Age (Col 1) for Row 1 (Bob) df.iloc[1, 1] = 33
‚öîÔ∏è loc vs iloc Showdown
Feature loc[] iloc[]
Type Label-based (Names) Position-based (Integers)
Slicing End ‚úÖ INCLUDED (['a':'c'] includes 'c') ‚ùå EXCLUDED ([0:2] excludes 2)
Conditional ‚úÖ Native (df.loc[df['A']>5]) ‚ö†Ô∏è Use .values
Inputs Strings, Lists, Labels Integers, Lists, Slices
