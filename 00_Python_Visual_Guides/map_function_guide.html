Python map() – Applying a Function to Every Item

The map() function applies a given function to each item of an iterable (like a list) and returns an iterator of the results.

Why use map()? It provides a concise, functional‑style way to transform collections without explicit loops.

Example: Squaring Numbers

def square(number):
    """Returns the square of a given number."""
    return number * number

# Input list
lst = [2, 5, 8, 9, 12, 13]

# Use map() to apply 'square' to every item in 'lst'
print("Output from the loop:")
for i in map(square, lst):
    print(i)

Running this script prints:

4
25
64
81
144
169

Notice that map() returns an iterator, so we can loop over it directly or convert it to a list with list(map(square, lst)) if we need random access.
