üõí Shopping Cart: Transforming Items

ANALOGY: Grocery Shopping with Price Doubling

You're shopping, but there's a special: Everything is 2x price today!
You need to calculate the new prices for all items in your cart.

üçé
Apple: $1
‚Üí Doubled: $2
üçå
Banana: $2
‚Üí Doubled: $4
üçä
Orange: $3
‚Üí Doubled: $6

The Code:

STEP-BY-STEP (For Loop)
# Manual calculation
prices = [1, 2, 3]
doubled = []

for price in prices:
    doubled.append(price * 2)

print(doubled) # [2, 4, 6]
ONE-LINE (List Comprehension)
# Instant calculation!
prices = [1, 2, 3]

doubled = [price * 2 for price in prices]

print(doubled) # [2, 4, 6]
üõí List Comprehension = Shopping Cart Scanner
Scan each item, apply the transformation, put in new cart - all in one motion!
