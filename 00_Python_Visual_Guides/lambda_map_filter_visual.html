‚Ä¢ Tests each item with a condition

üîó Combining: The Complete Factory

ANALOGY: Water Treatment Plant

üåä Dirty Water (Raw Data) [1, 2, 3, 4, 5, 6]
‚¨áÔ∏è FILTER: Remove Odd Numbers
üíß Filtered Water [2, 4, 6]
‚¨áÔ∏è MAP: Square Each Number
‚ú® Clean Water (Final Result) [4, 16, 36]
COMPLETE PIPELINE

# Step 1: Filter (keep only evens)

# Step 2: Map (square each number)

print(result) # [4, 16, 36]

# Or combine in one line:
