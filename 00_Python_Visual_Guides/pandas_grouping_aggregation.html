Mastering groupby, sum, mean, and agg

Introduction

Grouping and aggregation are powerful techniques that allow you to:

  • Group data based on specific columns.
  • Perform calculations like sum, average, count, etc., on grouped data.
  • Summarize large datasets efficiently.

Grouping helps in organizing data into categories, while aggregation helps calculate meaningful insights from grouped data.

Why Grouping and Aggregation are Useful
  • Efficient handling of large datasets.
  • Useful for data summarization and statistical analysis.
  • Helps in understanding patterns and relationships within data.
Key Functions
Function Description Example
groupby() Groups data based on values in one or more columns df.groupby('Category')
sum() Returns the sum of values for each group df.groupby('Category').sum()
mean() Returns the mean of values for each group df.groupby('Category').mean()
count() Returns the count of non-null values for each group df.groupby('Category').count()
max() Returns the maximum value for each group df.groupby('Category').max()
min() Returns the minimum value for each group df.groupby('Category').min()
agg() Allows multiple aggregation functions at once df.groupby('Category').agg(['sum', 'mean'])
Sample DataFrame
import pandas as pd data = { 'Product': ['Laptop', 'Phone', 'Tablet', 'Monitor', 'Keyboard', 'Mouse', 'Headphones', 'Phone', 'Tablet', 'Laptop'], 'Category': ['Electronics', 'Electronics', 'Electronics', 'Electronics', 'Accessories', 'Accessories', 'Accessories', 'Electronics', 'Electronics', 'Electronics'], 'Price': [1000, 500, 300, 150, 50, 20, 80, 600, 250, 1100], 'Stock': [10, 50, 30, 5, 100, 150, 50, 40, 20, 5] } df = pd.DataFrame(data) print(df)
1. Grouping Data (groupby)

Groups the data based on values in one or more columns.

grouped = df.groupby('Category') print(grouped)
2. Aggregation (sum)

Calculate sum of numerical columns for each group.

result = df.groupby('Category').sum() print(result)
Price Stock Category Accessories 150 300 Electronics 4250 155
3. Aggregation (mean)

Calculate the average value for each group.

result = df.groupby('Category').mean() print(result)
Price Stock Category Accessories 50.0 100.0 Electronics 708.3 25.83
4. Counting Values (count)

Count number of records in each group.

result = df.groupby('Category').count() print(result)
Product Price Stock Category Accessories 3 3 3 Electronics 7 7 7
5. Multiple Aggregations (agg)

Apply multiple functions at once.

result = df.groupby('Category').agg({'Price': ['sum', 'mean'], 'Stock': ['sum', 'mean']}) print(result)
Price Stock sum mean sum mean Category Accessories 150 50.000000 300 100.000 Electronics 4250 708.333333 155 25.833
6. Grouping on Multiple Columns

Group by Category AND Product.

result = df.groupby(['Category', 'Product']).sum() print(result)
Price Stock Category Product Accessories Headphones 80 50 Keyboard 50 100 Mouse 20 150 Electronics Laptop 2100 15 Monitor 150 5 Phone 1100 90 Tablet 550 50
