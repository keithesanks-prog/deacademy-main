Smoothing Data & Identifying Trends

What is a Rolling Average?

A moving average is the average of a fixed number of previous data points in a time series.

Key Benefits:
  • Smoothes out short-term fluctuations (noise).
  • Highlights longer-term trends.
  • Essential for forecasting sales, stock prices, etc.
Types of Moving Averages
Type Description Pandas Method
SMA (Simple Moving Average) Average over a fixed window (e.g., last 3 days). All points have equal weight. .rolling(window=n).mean()
EMA (Exponential Moving Average) Gives more weight to recent data points. Reacts faster to changes. .ewm(span=n).mean()
Syntax
# Simple Moving Average (Window = 3) df['SMA'] = df['column'].rolling(window=3).mean() # Exponential Moving Average (Span = 3) df['EMA'] = df['column'].ewm(span=3, adjust=False).mean()
Setup Data
import pandas as pd data = { 'Date': pd.date_range(start='2025-01-01', periods=10, freq='D'), 'Sales': [200, 220, 210, 215, 250, 300, 280, 270, 260, 310] } df = pd.DataFrame(data) df.set_index('Date', inplace=True) print(df.head())
1. Simple Moving Average (SMA)

Calculate average over a 3-day window.

df['Rolling_Avg'] = df['Sales'].rolling(window=3).mean() print(df[['Sales', 'Rolling_Avg']])
Sales Rolling_Avg 2025-01-01 200 NaN 2025-01-02 220 NaN 2025-01-03 210 210.000000 2025-01-04 215 215.000000 2025-01-05 250 225.000000 ... (values smooth out)

* Note: The first two rows are NaN because there isn't enough data for a window of 3.

2. Exponential Moving Average (EMA)

Weighted average that reacts faster to recent changes.

df['EMA'] = df['Sales'].ewm(span=3, adjust=False).mean() print(df[['Sales', 'EMA']])
Sales EMA 2025-01-01 200 200.000000 2025-01-02 220 210.000000 2025-01-03 210 210.000000 2025-01-04 215 212.500000 2025-01-05 250 231.250000
Window Size Impact
  • Small Window: Responsive, more noise.
  • Large Window: Smoother, slower lag.
