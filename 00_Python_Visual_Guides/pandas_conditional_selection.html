Pandas Conditional Selection üîç

Extract data based on specific criteria

üåü The Filtering Pattern

Ask a question (Condition). Get the answer (Rows).

df[ Condition ]
Example: df[df['Price'] > 100]

‚ùì The Question

df['Price'] > 50

Returns True/False list

üéØ The Filter

df[ ... ]

Keeps only True rows
1Ô∏è‚É£ Single Condition

Use standard mathematical operators: ==, !=, >, <. 

# Get all products with Price > 300
result = df[df['Price'] > 300]
print(result)
2Ô∏è‚É£ Multiple Conditions (AND / OR)
‚ö†Ô∏è Important Syntax: You MUST wrap each condition in parentheses (). 
# AND (&): Both must be true
# Price > 300 AND Stock < 20
result = df[(df['Price'] > 300) & (df['Stock'] < 20)]

# OR (|): At least one is true
# Price > 500 OR Stock > 40
result = df[(df['Price'] > 500) | (df['Stock'] > 40)]
üßæ isin() - The List Matcher

Perfect for checking if a value exists in a specific list (like SQL IN).

# Get only Electronics and Toys
result = df[df['Category'].isin(['Electronics', 'Toys'])]
‚ÜîÔ∏è between() - The Range Finder

Inclusive filtering of ranges.

# Price between 100 and 500
result = df[df['Price'].between(100, 500)]
üó£Ô∏è query() - The SQL Style

Write filters as a simple string. Often cleaner to read.

# Very readable syntax!
result = df.query('Price > 300 and Stock < 20')
üîç str.contains()

Filter based on text content (like SQL LIKE).

# Get all products with "Phone" in the name
result = df[df['Product'].str.contains('Phone')]
